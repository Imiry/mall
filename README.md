# mall类似于一个蘑菇街的手机app，前端使用vue进行开发，本项目只开发了首页和首页详情等，以及添加商品到购物车的功能，后期会完善分类模块和个人信息模块等

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
